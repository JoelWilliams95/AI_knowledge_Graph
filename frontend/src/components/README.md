Components:

- **SearchInterface.js** - Main search interface for papers and entities
  - Search papers by keywords
  - Browse available papers collection
  - View individual paper graphs
  - Search entities across all papers

- **UploadForm.js** - Optional file upload component
  - Upload new PDF papers
  - Process and add to collection

- **GraphViewer.js** - Interactive graph visualization
  - Cytoscape.js integration
  - Responsive design (desktop/mobile)
  - Node selection and expansion
